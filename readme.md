Versions used:

- Angular CLI: 14.1.2
- Node: 22.x
- Package Manager: npm 7.24.0


Commands to run:

- npm init
- npm install
- npm start (http://localhost:3000/)

Git:

- git add . && git commit -m "c" && git push origin master

Vercel: (app name: express-vercel-app)

- vercel --prod

All in one:

- git add . && git commit -m "c" && git push origin master && vercel --prod
